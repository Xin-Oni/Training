# クイックスタートガイド

ドクターヘリ物品チェックリストアプリを今すぐ始めるための最短手順です。

## 🚀 3ステップで始める

### ステップ1: Google Sheets を準備（5分）

1. [このテンプレート](https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms/copy)からコピーを作成
   - または、新しいシートを作成して `Resources/SpreadsheetTemplate.csv` をインポート

2. スプレッドシートのURLから**ID**をコピー：
   ```
   https://docs.google.com/spreadsheets/d/[ここがID]/edit
   ```

3. 共有設定を「リンクを知っている全員」に変更

### ステップ2: API キーを取得（3分）

1. [Google Cloud Console](https://console.cloud.google.com/) を開く
2. プロジェクトを作成
3. Google Sheets API を有効化
4. APIキーを作成してコピー

詳しくは → [SETUP_GUIDE.md](SETUP_GUIDE.md) の「ステップ3」参照

### ステップ3: アプリを起動（2分）

1. Xcode でプロジェクトを開く：
   ```bash
   open DoctorHeliChecklistApp.xcodeproj
   ```

2. ⌘+R でアプリを実行

3. 設定画面（⚙️）を開いて：
   - スプレッドシートID を入力
   - API キー を入力
   - 「保存」をタップ

4. 完了！🎉

## 📱 基本的な使い方

### チェックする
- ○アイコンをタップ → ✓に変化
- 自動的にスプレッドシートに同期

### 使用期限を設定
1. アイテムをタップ
2. 「使用期限を設定」をON
3. 日付を選択
4. 「保存」

### 新しい物品を追加
1. ＋アイコンをタップ
2. 情報を入力
3. 「追加」

### カテゴリで絞り込み
- 画面上部のボタンをタップ

### 検索
- 上部の検索バーに入力

## 🎯 重要なポイント

✅ チェックすると自動で同期
✅ オフラインでも動作（次回接続時に同期）
✅ 期限切れは赤色表示
✅ 期限間近（30日以内）はオレンジ色表示

## 🆘 困ったときは

**同期できない？**
- スプレッドシートの共有設定を確認
- 🔄アイコンで手動同期を試す

**データが表示されない？**
- スプレッドシートの形式を確認（A列=物品名、B列=カテゴリ...）
- 1行目はヘッダー行
